ArrayList

  • a collection class

Creating ArrayList Objects

Constructor Description
ArrayList<Type>() Creates an empty array list with an initial capacity to hold 10 objects
ArrayList<Type>(int n) creates an empty ArrayList with an initial capacity of n objects of the specified type.
ArrayList<Type>(Collection obj) creates an ArrayList that contains the elements of another Collection object

ArrayList Methods

Method Description
add(object) adds the specified object to the end of the ArrayList
add(index, object) adds the specified object to the ArrayList at the specified index
set(index, object) sets the element at the specified index to a specified object
remove(index) removes the element at the specific index
get(index) returns the element at the specific index
size() returns the number of objects in the ArrayList
isEmpty() returns true if the ArrayList is empty
clear() Clears the ararylist
