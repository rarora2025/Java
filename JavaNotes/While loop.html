Java While Loop

The while loop loops through a block of code as long as a specified condition is true:

Syntax

while (condition) {
  // code block to be executed
}

UML Activity Diagram for a while statement

The Merge and Decision Symbol

  • both represented in diamonds
  • connected with arrow

Decision Symbol- has one transition arrow pointing to the diamond and two or more pointing from it to indicate possible transition from that point. Merge Symbols have labels (guard statements) on transition arrows out from it.

Merge Symbol- has two or more transition arrows pointing to the diamond and only one pointing from the diamond, to indicate multiple activity flows merging to continue the activity. Merge Symbols have no labels (guard statements) on arrows out from it.

Sentinel Values

  • used to indicate the end of a data entry
  • For instance, the user can type this sentinel value to exit the program
  • Sentinel-controlled interation (indefinite iteration) iteration that will occur until a sentinel/flag/signal value is entered by the user; the number of iterations is not known before the loop begins executing

Implicit vs Explicit Conversion

Implicit- done automatically by the computer, for example if you divide an int by a double then the int will automatically be converted to a double

Explicit- type casting, placing the unary cast operator (double) before a statement to convert it
