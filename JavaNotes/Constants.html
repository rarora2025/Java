Constants

constants- variables whose value cannot be changed once it is initialized

Benefits: protects the variable from inadvertent (careless) errors

How to declare a constant?

  • add the keyword final at the beginning of the variables declaration statements

*Constants can only be declared once- either in declaration or in an assignment statement

*Good Programming Practice- it is a good programming practice to name constants in ALL CAPS; it helps the variables stand out visually and lets readers of the program know that they are constants

Example of Usage:

final int APPLES = 10;

Example of Error:

  final int BANANAS = 10;
    BANANAS = 99
