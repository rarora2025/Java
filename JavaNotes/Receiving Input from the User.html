Receiving Input from the User

  • a great strength of Java is its rich set of predefined classes that you can reuse rather than “reinventing the wheel”; these classes are grouped into packages
  1. The first thing you need to do to get input from the user, is to import the package/class ‘Scanner’

import java.util.Scanner;

  1. Next, define a Scanner variable

Scanner input = new Scanner(System.in);

  1. Prompt the User for Input- you can do this by writing a simple println statement-

System.out.println("Enter first integer ");

  1. Declare a variable to store and obtain the input-

if the input is an int declare like this:

int users_int = input.nextInt();

otherwise declare like this:

int users_input = input.nextLine();
