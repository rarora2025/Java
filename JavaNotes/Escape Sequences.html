Sequence Description
\b (backspace) Moves screen cursor one character to the left
\t (tab) Moves screen cursor to the next tab position
\n (linefeed) Moves screen cursor to the beginning of the next line
\r (carriage return) Moves screen cursor to beginning of the current line
\\ (backslash) Used to display the backslash () character
\' (single quote) Used to display a single quote character
\" (double quote) Used to display a double quote character
