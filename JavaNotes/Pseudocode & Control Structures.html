Pseudocode

  • an informal language that helps you develop algorithms withou having to worry about the strict details of language syntax
  • does not executed on computers
  • it helps you think out a program before attempting to write in a programming language

The 3 Control Structures

There are Control Statements:

  1. The sequence structure
  2. The selection structure
  3. the iteration structure

The Sequence Structure

  • built in java; computer executes statementes in order listed (top to bottom)

The Selection Structure

  • three types:
  1. if statements
  2. if-else statement
  3. switch statement
  • three different types of selection statements
  1. single-selection statement (if statement)
  2. double-selection statement (if-else statement)
  3. multiple-selection statment (switch statement)

The Iteration

  • four different statements; also called repitition/looping statements
  • we will learn about these statements later
  1. while
  2. do...while
  3. for
  4. enhanced for
