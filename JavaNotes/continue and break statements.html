Continue and Break statements

  • In a while, for, or do…while, these keywords can be used

The Break Statement

  • exits/terminates the loop entirely

Example

for (int i = 0; i < 10; i++) {
  if (i == 4) {
    break;
  }
  System.out.println(i);
}
//All numbers up to four will be printed

The Continue Statement

  • skips the remainding statements in the loop body and proceeds with the next iteration of the loop

Example

for (int i = 0; i < 10; i++) {
  if (i == 4) {
    continue;
  }
  System.out.println(i);

}

//all numbers BUT four will be printed
