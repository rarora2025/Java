Machine, Assembly and High-Level Languages

Machine Languages

  • generally consists of strings of numbers that instruct computers to perform their most elementary operations one at a time
  • these are machine dependent

Assembly Languages

  • languages that programmers understand that are translated to machine languages so that the computer can read them
  • much more clear to humans, but are incomprehensible to computers until translated

Assemblers

  • translator programs developed to translate from Assembly Languages to Machine Languages

High-Level Languages

  • Even more comprehensible to programmers, and a lot more efficient
  • Also translated to Machine Language for the computer to read (using Compilers)

Compilers

  • programs that convert high-level language programs into machine language very fast
