5 Stages of Development

Alpha- earliest release of a software product that’s still under active development

Beta- released to larger number of developers later in the development process after most major bugs have been fixed and new features are almost complete

Release Candidates- generally feature complete, bug free, and ready for use by the community

Final release- All bugs from release candidate are corrected and the final product is released to the general public

Continuous beta- Software that’s developed using this approach generally does not have version numbers. It is hosted in the cloud (servers over internet) and is constantly evolving so that users always have the latest version
