Logical Units of a Computer

Logical Unit Description
Input unit receives the information, mostly through mouse, keyboard, camera, etc.
Output Unit takes the information that the computer has processed and ships it out/displays it
Memory Unit “the warehouse”; retains information that has been entered through the input unit
Arithmetic and Logic Unit (ALU) “manufacturer”; performs calculations, and contains decision mechanisms
Central Processing Unit (CPU) “administrator”; coordinates and supervises the operation of the other sections
Secondary Storage Unit “long-term warehouse”; contains the programs/data that is not actively being used by other units
