The Selection Sort Algorithm- O(n^2) Complexity

The selection sort algorithm is a very simple sort algorithm. Suppose we have a list of numbers that we want to sort in ascending order. In selection sort algorithm, we search for the lowest element and arrange it to the proper location. We swap the current element with the next lowest number.

How does selection sort work?

The selection sort algorithm works in a very simple way. It maintains two subarray for the given array.

  • The subarray is already sorted.
  • And the second subarray is unsorted.

With every iteration of selection sort, an element is picked from the unsorted subarray and moved to the sorted subarray.

For example:

We start with an unsorted list, and [] select the smallest value in the list. We then place this value first in the list and swap it with the value that was originally there.
Now, [] the number 1 is in the correct position, so it needs no further consideration.
Next, [] we select the smallest value in the remaining list, which is 2, and swap it with the value currently in the second position.
Now, [] the number 2 is in the correct position and no longer needs to be considered.
Next, [] we select the smallest number in the remaining list, which is 4, and swap it with the value currently in the third position.
Now, [] 4 is in the correct position and no longer needs to be considered.
Next, [*] we select the smallest value in the remaining list, which in this case is 5, and since it’s already in the correct position no swap is necessary.

public  static  void  selectionSort(int[]  arr)

{

// run (arr.length - 1) times

for  (int  i  =  0;  i  <  arr.length  -  1;  i++)

{

// find the minimum element in the unsorted sub-array[i..n-1]

// and swap it with arr[i]

int  min  =  i;

for  (int  j  =  i  +  1;  j  <  arr.length;  j++)

{

// if arr[j] element is less, then it is the new minimum

if  (arr[j]  <  arr[min])  {

min  =  j;  // update index of min element

}

}

// swap the minimum element in sub-array[i..n-1] with arr[i]

swap(arr,  min,  i);

}

}

Accomplishing the Selection Sort Recursively

// Recursive function to perform selection sort on sub-array arr[i..n-1]

public  static  void  selectionSort(int[]  arr,  int  i,  int  n)

{

// find the minimum element in the unsorted sub-array[i..n-1]

// and swap it with arr[i]

int  min  =  i;

for  (int  j  =  i  +  1;  j  <  n;  j++)

{

// if arr[j] element is less, then it is the new minimum

if  (arr[j]  <  arr[min])  {

min  =  j;  // update index of min element

}

}

// swap the minimum element in sub-array[i..n-1] with arr[i]

swap(arr,  min,  i);

if  (i  +  1  <  n)  {

selectionSort(arr,  i  +  1,  n);

}

}

The Insertion Sort Algorithm- O(n^2) Complexity
The insertion sort algorithm sorts a list by repeatedly inserting a new element into a sorted sublist until the entire list is sorted.

Let’s visually walk through the insertion sort for this array of integers.
Intially, we use the first value in the array as the first value in the sorted sublist.
Then, as a first step, we take the second number on the list, [] in this case the number 9, and insert it in ascending order into the existing sublist.
[].The sorted sublist is now 2 and 9.
Next, [] we take the number 5 and insert it into the sublist in ascending order.
The sorted sublist is now 2, 5, 9.
As our third step, [] we insert 4 into the sorted sublist.
[] The sorted sublist is now 2, 4, 5, 9.
Next, [] we insert 8 into the sorted sublist.

public static void insertionSort(int array[]) {
    for (int j = 1; j < array.length; j++) {
        int current = array[j];
        int i = j-1;
        while ((i > -1) && (array[i] > current)) {
            array[i+1] = array[i];
            i--;
        }
        array[i+1] = current;
    }
}

Accomplishing the Insertion Sort Recursively

public static int[] InsertionSort(int[] array, int n) 
{
    int i;
    if (n < 1) {
        InsertionSort(array, n - 1);
    } 
    else {

        int key = array[n];
        i = n - 1;

        while (i >= 0 && array[i] > key) {
            array[i + 1] = array[i];
            i = i - 1;
        }

        array[i + 1] = key;
    }

    return array;
}
