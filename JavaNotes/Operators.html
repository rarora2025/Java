Operators

Binary Operators- operators that operate two operands, such as

  • + (addition)
  • - (subtraction)
  • / (division)
  • * (multiplication)
  • % (modulo/remainder)

Integer division always yields an integer quotient

Equality Operators- operators that check if their operands are equal, such as

  • == (is equal to)
  • != (is not equal to)

Relational Operators- operators that compare their operands, such as

  • < (is less than)
  • > (is greater than)
  • <= (is less than or equal to)
  • >= (is greater than or equal to)

If statements- evaluate a condition

Syntax:

if (condition) {
//run this code
}

Are Curly Braces Required?
–do you NEED curly braces: no! (if you have one line of code)

Advantage of omitting {}: Your code can be briefer - meaning less line. Some may feel it is more readable.

Disadvantage of omitting {}: If you later modify your code and need to add another line to the code block, you better remember to add those {}.
