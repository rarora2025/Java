Logical AND/OR Operator vs Conditional AND/OR

AND

Unlike the conditional AND Operator (&&), the logical AND operator evaluates both expressions, no matter wether or not the first one is true or false. This helps in certain situation for example:

if (x > 10) & (++x < 10) { }
because you want x to increment no matter what

OR

Unlike the conditional OR operator (||), the logical (exclusive) OR operator (^) will retry true if and only if one of its operands is true and the other is false. If both are true or both is false then the entire conditional is false
