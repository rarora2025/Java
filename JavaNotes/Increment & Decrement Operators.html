Incrementing and Decrementing Variables in Java

these operators are used to subtract or add one to a variable

Increment Operator: ++
Decrement Operator: --

prefix vs posftix operators:

  • prefix comes before the variable name, postfix comes after; So for example

x++ would be a postfix increment.

Table of Operators

Operator Name Sample Expression Explanation
++ prefix increment ++x Increment x by 1, then use the new value of x in the expression in which x resides
++ postfix increment x++ Use the current value of x in the expression in which x resides, then increment x by 1.
-- prefix decrement --x Decrement x by 1, then use the new value of x in the expression in which x resides
-- postfix decrement x-- Use the current value of x in the expression in which x resides, then decrement x by 1.

preincrementing (/predecrementing) vs
postincrementing (/postdecrementing)

examples:
if you want to print the value of a number BEFORE it increments/decrements, use postincrementing (x++/x–)

– if c==5 and you do
System.out.println(c++)
result will be 5

otherwise, if you want to print the value of a number after it has already incremented/decremented, use pretincrementing (++x/–x)

– if c==5 and you do
System.out.println(++c)
result will be 6
