Containment

the idea that an outer class contains an instance of another class and allows access to the contained object through its own method

  • a “has-a” relationship, also part-whole

When containment is used, the containing class must take responsibility for ensuring that the contained class constructors are properly invoked:

For example, in the default constructors or in constructors in which certain attributes are not specified and are given default values, if any of the attributes are objects of a class, make sure to create new objects of a class (with appropriate arguments) in those constructors.

UML- to represent containment, draw an arrow with the word “uses” on it, and make it pointing to the class whose object is being “contained”

For example, if we have a person class that contains objects of class date, name, and age, then draw “uses” arrows from the person class to each of the class diagrams of name, date, and age.

Aggregation

Indicated by a straight line with a clear diamond shape at the aggregating class in UML Symbols

[the aggregating class is the class that is aggregating/containing the other ‘aggregated’ classes]

Aggregating = whole
Aggregated = part

  • a type of containment
  • In aggregation, the contained object can exist independently from the outer object
  • an aggregated object is not created within the “container” class, but instead it is created outside and then later assigned to a reference class (until it is assigned, its default value may be null)

Example: a Family object might contain an instance of a Dog object, representing a dog that the family owns.

Composition

Indicated by a straight line with a darkened diamond shape at the aggregating class in UML Symbols

  • a type of containment
  • just a stronger form of aggregation
  • In composition, the contained object is created and exists entirely within the object that contains it: It ceases to exist when the containing object is destroyed.
  • also known as a “part-of” relationship

Example: water and pond: water is A PART OF a pond
