System.out.printf()

System.out.printf prints text in a given format
It takes in format parameter, and then the number of additional arguments you’ve specified.

For example:

System.out.printf(%s%n%s%n, "Hello", "Rahul")
//prints: Hello Rahul on two different lines

the %s%n… is the format, and each format specifier is a placeholder for a value and specifies the type of data to output.

For instance %s is a String placeholder and %n is a new line
