Bits

  • the smallest data item in a computer
  • can assume the value 0 or 1
  • “bit” is short for a “binary digit”

Bits to Bytes

  • there are eight bits in one byte

Characters

  • composed of bits
  • includes decimal digits, letters, and special symbols
  • Java uses Unicode characters
  • “bit” is short for a “binary digit”

Fields

  • composed of characters
  • a group of characters that conveys meaning, like a persons name or age

Records

  • a group of related fields
  • implemented as a class in java

Files

  • group of related records
  • contains arbitrary data in arbitrary formats

Database

  • collection of data for easy access and manipulation

  • most popular model is the relational database
    - Data is stored as tables, which contain records and fields

  • You can search, sort and manipulate the data based on its relationship to multiples tables/databases

Big Data

  • types of applications that deal with massive amounts of data
